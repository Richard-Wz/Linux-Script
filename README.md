## Script

### Update Script (`update.sh`)
Automates Ubuntu system updates including apt update, upgrade, autoremove, and autoclean.

## Usage

```bash
git clone https://github.com/Richard-Wz/Linux-Script.git
cd Linux-Script
chmod +x update. sh
sudo ./update.sh
```

## Requirements

- Ubuntu/Debian-based Linux distribution
- Root/sudo privileges
- Bash shell
